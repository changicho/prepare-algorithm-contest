## 설계

### 시간 복잡도

입력받은 두 수로 연산을 수행하는데 O(1)의 시간 복잡도를 사용한다.

### 공간 복잡도

연산에 필요한 변수만 사용하므로 O(1)의 공간 복잡도를 사용한다.

### 연산

| 내 코드 (ms) | 시간 복잡도 | 공간 복잡도 |
| :----------: | :---------: | :---------: |
|      1       |    O(1)     |    O(1)     |

```cpp
long long solution(int w, int b) {
  w *= 1000;

  return ceil(w / b) + 1;
}
```
