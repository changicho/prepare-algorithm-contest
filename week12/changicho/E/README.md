## 설계

### 시간 복잡도

원소의 갯수를 N이라 하자.

그리디 알고리즘을 이용해 배열을 생성할 경우 O(N)의 시간 복잡도를 사용한다.

### 공간 복잡도

정답을 배열로 반환할 경우 O(N)의 공간 복잡도를 사용한다.
